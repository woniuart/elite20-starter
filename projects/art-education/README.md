# 艺术通识课AI教学助手

## 项目简介
针对大学公共艺术选修课AI落地的教学助手矩阵

## 目录结构
```
projects/art-education/       # 核心项目内容
├── docs/                     # 文档（KSTAR、坐标卡、开发日志）
├── prompts/                  # AI提示词库
├── outputs/                  # 教学成果（预习单等）
└── scripts/                  # 自动化脚本

Starter-Kit/                  # C2S完整提交包
├── 赵洁_C2S_方案草案.md
├── 赵洁_C2S_方案设计.md
├── 赵洁_C2S_AI日志.md
├── 赵洁_C2S_拿来说明.md
├── 赵洁_C2S_G2验证报告.md
└── projects/art-education/   # 链接到核心项目
```

## 核心功能
- 美术鉴赏知识库
- 课前预习助手
- 作业批改助手
- 课后复习助手

## 快速开始
1. 克隆仓库：`git clone https://github.com/woniuart/elite20-starter.git`
2. 查看项目：`cd projects/art-education`
3. 使用提示词：参考 `prompts/` 目录
