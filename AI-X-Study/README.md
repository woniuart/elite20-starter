## 🔑 核心概念

### K-S-T-A-R 循环
- **K - 知识**：需要什么信息？
- **S - 情境**：具体情况/约束是什么？
- **T - 任务**：真正要完成的是什么？
- **A - 行动**：做了什么（包括提示词）？
- **R - 结果**：发生了什么？是否符合预期？

### AI+X 三维框架
- **X轴**：你的领域（专业/语言/品味/问题）
- **AI轴**：你的杠杆（提示词/Skills/工具/工作流）
- **S轴**：你的自我（反思/作品集/K-S-T-A-R复盘）
