📝 提示词示例 1：微积分求极限

场景：需要 AI 智能体帮忙分析一道大一的微积分求极限题目，要求有答案解析、分步讲解、易错点总结，让零基础的学生能学会理解。

提示词：

你现在是微积分助教，帮我拆解下面这道求极限题，要求：
1. 先写出题目，再分步讲解每一步的解题思路
2. 标注出这道题的易错点和常见坑
3. 最后给一个通用的解题模板，方便我套用在同类题目上
4. 语言要口语化，不要用太生硬的书面腔

题目：求 lim(x→0) (sinx - x)/x³

AI 输出：

哈喽同学！这道极限题是大一微积分的高频考点，我给你拆解得明明白白👇

🔹 题目还原：求极限 lim(x→0) (sinx - x)/x³

🔹 分步解题思路：

1. 第一步：先判断能不能直接代入 - 出现0/0型不定式，需要用洛必达法则或泰勒公式

2. 第二步：用泰勒公式展开 - sin x = x - x³/6 + o(x³)

3. 第三步：代入原式化简 = -1/6

🔹 易错点提醒：❌ 错误做法：直接用等价无穷小替换 sin x ~ x，这样会得到0，这是错的！✅ 正确做法：加减法里用泰勒公式展开到x³项

🔹 同类题通用模板：1. 先判断分母的阶数n 2. 把分子里的三角函数用泰勒公式展开到n阶 3. 代入化简，约掉公共项

轨迹日志：

维度内容
清晰度✅ 明确要求分步讲解、易错点、通用模板
上下文✅ 设定了"微积分助教"角色身份
约束条件✅ 4条具体要求
期望格式✅ 要求口语化输出

📝 提示词示例 2：Python 代码 Debug

场景：想让 AI 智能体帮忙检查 Python 代码里是否有 bug，并给出修改后可以正常运行的代码，同时告诉错误原因。

提示词：

帮我看看这段Python代码有什么问题，要求：
1. 先指出错误原因，用大白话解释
2. 给出修改后的完整可运行代码
3. 加一行注释说明修改的地方

代码：
def add_numbers(a, b):
    result = a + b

print(add_numbers(2, "3"))

AI 输出：

🔍 错误原因：

1. 函数里没有用return返回结果，调用函数时拿不到计算结果，会默认返回None

2. 你把数字2和字符串"3"相加，Python不支持数字和字符串直接相加，会报TypeError错误

修改后的代码：

def add_numbers(a, b):
    # 把输入的参数都转成整数，避免类型不匹配的问题
    result = int(a) + int(b)
    # 加上return，把计算结果返回给调用者
    return result

print(add_numbers(2, "3"))  # 输出: 5

轨迹日志：

维度内容
清晰度✅ 分三条要求清晰列出
上下文✅ 提供具体代码片段作为上下文
约束条件✅ 要求解释+代码+注释三点
期望格式✅ 要求大白话解释，代码要完整可运行

📝 提示词示例 3：学术论文翻译

场景：从一篇关于 AI+X 教育范式、共生智能、AI 智能体的英文学术论文中，自动提取最核心、最有研究价值的一段，进行精准学术翻译，并标注关键术语。

提示词：

你是AI智能体与教育领域的学术翻译助手。请从我提供的英文论文中，自动筛选论文最核心、最有价值的一段，只翻译这一段。
要求：
1. 译文符合中文学术规范，严谨通顺。
2. 标出本段3–5个核心专业术语并简要解释。
3. 用一句话概括本段核心观点。

AI 输出：

原文：Symbiotic Intelligence can be defined as a collaborative cognitive ecosystem in which human and artificial intelligences mutually enhance each other to achieve goals neither could attain alone...

精准中文翻译：共生智能可被定义为一种协作式认知生态：人类智能与人工智能相互增强，共同完成任何一方单独无法实现的目标。它是一种协同创造范式，而非竞争范式...

核心术语解释：

  • Symbiotic Intelligence 共生智能：人机双向增强、互补协作的新型智能形态
  • Cognitive ecosystem 认知生态：由人与AI共同构成的协同思考与决策系统
  • Pattern recognition 模式识别：AI从大量数据中快速发现规律的核心能力

一句话核心观点：本段定义了共生智能，明确人机不是竞争而是互补协作，通过合理分工形成更强大的统一认知单元。

轨迹日志：

维度内容
清晰度✅ 三条要求明确分工（翻译、术语、总结）
上下文✅ 设定了学术翻译助手角色
约束条件✅ 规定术语数量（3-5个）、语言风格（学术规范）
期望格式✅ 分段输出（译文+术语+观点）

💭 反思

提示工程学习总结：通过三个不同场景的练习，深刻体会到好提示词的核心要素：1）角色设定帮助 AI 理解身份；2）具体任务描述清晰目标；3）约束条件明确输出范围；4）格式要求便于阅读使用。提示词工程是人与 AI 高效协作的关键技能。
