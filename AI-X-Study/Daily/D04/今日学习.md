## 📝 提示词示例 1

**场景**：需要 AI 智能体帮忙分析一道大一的微积分求极限题目，要求有答案解析、分步讲解、易错点总结，让零基础的学生能学会理解。

**提示词**：
```
你现在是微积分助教，帮我拆解下面这道求极限题，要求：
1. 先写出题目，再分步讲解每一步的解题思路
2. 标注出这道题的易错点和常见坑
3. 最后给一个通用的解题模板，方便我套用在同类题目上
4. 语言要口语化，不要用太生硬的书面腔

题目：求 lim(x→0) (sinx - x)/x³
```

**AI 输出**：
```
哈喽同学！这道极限题是大一微积分的高频考点，我给你拆解得明明白白👇

🔹 题目还原：求极限 lim(x→0) (sinx - x)/x³

🔹 分步解题思路：

1. 第一步：先判断能不能直接代入
把x=0代入分子：sin0 - 0 = 0；分母：0³=0，出现了0/0型不定式，不能直接算，需要用洛必达法则或者泰勒公式。

2. 第二步：用泰勒公式展开（推荐，比洛必达更稳）
我们知道sin x在x=0处的泰勒展开式：
sin x = x - x³/6 + o(x³)

3. 第三步：代入原式化简
把展开式代入分子：sin x - x = (x - x³/6) - x = -x³/6
原式就变成了：lim(x→0) (-x³/6)/x³ = -1/6

🔹 易错点提醒（很多同学都踩过的坑！）

❌ 错误做法：直接用等价无穷小替换 sin x ~ x，这样会直接得到 (x-x)/x³=0，这是错的！因为等价无穷小替换只能在乘除法里用，加减法里用会丢失高阶项的信息。

✅ 正确做法：加减法里用泰勒公式展开，展开到和分母同阶的项。

🔹 同类题通用模板
遇到0/0型的三角函数/多项式极限，且是加减法形式时：
1. 先判断分母的阶数n
2. 把分子里的三角函数/对数用泰勒公式展开到n阶
3. 代入化简，约掉公共项，就能直接算出结果啦！
```
