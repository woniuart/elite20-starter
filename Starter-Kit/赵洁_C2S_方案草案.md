# 赵洁_C2S_方案草案

## 1. 我打算做什么？解决什么具体问题？

针对大学艺术通识选修课AI落地无规范、教师不会设计AI+艺术课、学生滥用AI的痛点，基于Elite20三阶段体系，完成艺术学科化升级的基础实践：
- 搭建个人艺术教学智能体环境
- 开发4类艺术教学AI助手（美术鉴赏知识库、课前预习、作业批改、课后复习）
- 完成一次完整的艺术教学K-STAR闭环
- 产出可直接用于下学期课程的《蒙娜丽莎》课前预习AI助手

## 2. 我凭什么觉得这个方向有效？

- 理论依据：Elite20 AI+X三维目标（L1/L2/L3）、双回路教学法、KSTAR框架
- 实践基础：有10年公共艺术教学经验，熟悉全校艺术选修课大纲和学生痛点
- 参考案例：西亚斯学院Elite20艺术学科试点项目，已验证AI+艺术课可提升学生参与度40%

## 3. 我拿到的Token/算力打算怎么花？

- 预估总消耗：≤30元（全部用Claude免费额度+自有API额度，不申请额外Token）
- 用途：生成教学提示词、开发预习助手、验证作业批改功能
- 实验次数：共3次，分别测试提示词效果、内容准确性、学生易用性

## 4. 失败了怎么办？

- 若AI生成内容不符合教学要求：人工修正并优化提示词，保留迭代过程作为学习证据
- 若工具调用失败：改用Claude桌面端手动完成，记录工具问题和替代方案
- 最低可接受成果：完成环境搭建+3次GitHub提交+1次K-STAR闭环，达到Level1标准
