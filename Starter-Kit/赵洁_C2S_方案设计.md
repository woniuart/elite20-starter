# 赵洁_C2S_方案设计（正式版）

## 项目概述
- **项目名称**：艺术通识课AI教学助手
- **负责人**：赵洁
- **课程**：大学艺术通识选修课（美术鉴赏）
- **时间**：2026年4月30日

## 问题诊断

### 现状分析
- 教师：不熟悉AI工具，难以设计AI+艺术课
- 学生：滥用AI生成作业，学术诚信风险
- 教学：缺乏AI时代艺术教育规范

### 目标定位
基于Elite20三阶段体系构建艺术学科AI教学助手矩阵

## 解决方案

### 核心产出
1. **美术鉴赏知识库** - 艺术家生平、作品分析、艺术史脉络
2. **课前预习助手** - 知识点推送、思考题、背景资料
3. **作业批改助手** - AI检测、反馈建议、改进指导
4. **课后复习助手** - 知识巩固、创作灵感激发

### 本学期重点
《蒙娜丽莎》课前预习AI助手 - 完整可用的教学智能体

## 实施计划

| 周次 | 任务 |
|------|------|
| Week1 | 环境搭建 + 知识库基础框架 |
| Week2 | 开发《蒙娜丽莎》预习助手 + 实验1 |
| Week3 | 实验2 + 实验3 + 反馈迭代 |
| Week4 | 成果整理 + 验收 + 复盘 |
