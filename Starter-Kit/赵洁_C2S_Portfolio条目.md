## 项目简介

针对大学公共艺术选修课AI落地无规范的问题，开发了一套艺术教学AI助手矩阵，包括：
- 美术鉴赏知识库生成系统
- 课前预习AI助手
- 作业批改辅助工具
- 完整的学习资源体系

---

## 核心技术

- **AI工具**：Claude 3 Opus + GitHub API
- **开发方式**：Prompt Engineering + API集成
- **版本管理**：Git + GitHub
