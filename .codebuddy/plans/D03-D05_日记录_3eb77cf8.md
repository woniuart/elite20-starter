## 技术方案

### 文件格式

- 源文件：Markdown (.md)
- 导出格式：HTML（用于打印为 PDF）

### 实现步骤

1. 创建 D03~D05 的 Markdown 源文件，参照 D02 模板结构
2. 使用 md_to_html.py 脚本生成 HTML 文件
3. 预览并指导用户通过浏览器打印为 PDF

### 模板结构

每个日记录包含：

- 标题 + 日期
- 今日交付物清单
- 主题相关内容（根据课程大纲填充）
- 命令/操作记录
- 反思总结

### 关键脚本

- `md_to_html.py`：将 Markdown 转换为带样式的 HTML
